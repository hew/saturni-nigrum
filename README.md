# Occult Pyramid

A mystical 3D web experience built with Svelte and Three.js.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Technologies

- Svelte 5
- Three.js
- Vite
- GSAP
